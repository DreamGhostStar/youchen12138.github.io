栅格布局

将屏幕分成12格

  • 元素所占屏幕大小：col-屏幕型号-num

  • 元素距左侧几格：col-屏幕型号-offset-num

网页开发中的单位

px：相对于屏幕分辨率的长度单位

em：相对于当前文本内的字体尺寸

rem 与 em 相似，相对于 HTML 根节点的字体单位

px

px 是相对于屏幕分辨率的单位

px 的大小无法跟随屏幕放大缩小

em

1em = 16px（但不完全是）

em 会继承父级元素的字体大小，适合手机端、平板开发

IE 的部分版本不支持 em

rem

rem 与 em 类似

rem 会继承根元素的字体大小

bootstrap 组件

  1. role：标识普通标签，使之语义化，方便浏览器对其具体的功能进行识别

  2. aria-label：当焦点落到输入框时，读屏软件就会读出输入框内容

  3. tabIndex：可以设置键盘上的 tab 键在控件中进行移动

  4. 通过该自定义属性管理交互、效果

字体图标

  1. glyphicon

  2. glyphicon-具体图标

下拉菜单

  1. .dropdown 控制组件为下拉

  2. .dropdown-menu-right 代替 .pull-right 右对齐

  3. .divider 分割线

控件组

  1. .input-group 表示控件组

  2. .input-group-addon 可放置额外内容和图标

导航

  1. 以一个带有 class .nav 的无序列表开始

  2. .nav-tabs 代表可切换的导航

  3. .nav-pills 代表胶囊导航

  4. .nav-justified 使导航垂直

分页

  1. .pagination 在父元素中添加表示分页

  2. .pager 放置在翻页区域

  3. .previous 把链接向左对齐，.next 把链接向右对齐

进度条

  1. .progress 表示进度条

  2. 通过状态类改变进度条的颜色

  3. .progress-bar-striped 使得进度条颜色渐变

列表

  1. .list-group 代表列表组

  2. .badge 代表状态数

  3. .active 代表选中状态

面板

  1. .panel 代表面板

  2. .panel-body 代表面板内容

  3. .panel-footer 代表面板的注脚

Bootstrap 中的插件

  1. Bootstrap 插件依赖 Bootstrap.js

  2. Bootstrap.js 基于 JQuery

data 属性

  1. 通过 data 属性控制页面交互

  2. $(document).off(‘.data-api’) 解除属性绑定
