注意

  • 如果用 alert 或 document.write 输出不是字符串的值，则该值则调用 toString() 方法

  • document.write 期望接收和输出字符串，当接收的参数不为字符串时，会调用参数的toString方法，将其转化成字符串输出

  • 数组调用toString()后，会将其中的元素用逗号拼接起来变成字符串

  • 一般的对象调用toString()后返回[object Object]
