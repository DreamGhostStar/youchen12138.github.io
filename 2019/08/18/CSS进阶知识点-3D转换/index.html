CSS进阶知识点--3D转换及perspective 属性

3D转换

  • rotate3d()

  • translate3d()

  • scale3d()

  • perspective

旋转 rotateX/ rotateY / rotateZ

指定对象围绕着 X / Y / Z 轴上的旋转角度

语法

transform：rotateX / rotateY / rotateZ( angle )

旋转 rotate3d

指定对象的 3D 旋转角度

语法

transform：rotate3d(x, y, z, angle)

取值说明

前三个参数分别代表旋转的方向 x，y，z，第四个参数表示旋转的角度，参数不允许省略

前三个参数只有三个取值效果：正数，0，负数

如果多个方向发生改变，小数会代表在这个方向的旋转程度，最小小数取值为 0.1

移动 translateY

指定对象 Z 轴的平移（更多用于遮罩）

语法

transform：translateZ(z)

数值说明

z 值不能省略

移动 translate3d

指定对象的 3D 位移

语法

transform：translate3d(x, y, z)

数值说明

第一个参数对应 X 轴，第二个对应 Y 轴，第三个对应 Z 轴，参数不允许省略

缩放 scaleZ

指定对象的 Z 轴缩放

语法

transform: scaleZ(z)

参数说明

参数对应 Z 轴，不允许省略

缩放 scale3d

指定对象的 3D 缩放

语法

transform: scale3d(x，y，z)

参数说明

参数不允许省略

perspective 属性

指定观察者与 XOY 平面（也就是屏幕）的距离，使具有三维位置变换的元素产生透视效果

语法

perspective：number | none( 默认值 )

perspective-origin 属性

指定透视点的位置

语法

perspective-origin: x-axis y-axis

默认值

x-axis：50%

y-axis：50%
