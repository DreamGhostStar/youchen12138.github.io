CSS基础知识点--float浮动

CSS定位机制

  • 普通流（标准流）

  • 浮动

  • 绝对定位

普通流

普通流（标准流）：默认状态，元素自动从左到右，从上往下的排列

CSS Float（浮动）的基础知识

  • 会使元素向左或向右移动，只能左右，不能上下

  • 浮动元素碰到包含框或另一个浮动框，浮动停止

  • 浮动元素之后的元素将围绕它，之前的不受影响

  • 浮动元素会脱离标准流

浮动的基本语法

使用浮动后产生的问题

元素使用浮动后会脱离普通流，出现“高度塌陷”，也叫做浮动溢出。

清除浮动语法

clear：none | left | right | both

为什么要清除浮动

设置了float的元素会影响其他相邻元素，需要使用clear清除浮动，clear只会影响自身，不会对其他相邻元素造成影响！

清除浮动的常用方法

方法一：在浮动元素后使用一个空元素

例如：<div class="clear></div>

方法二：给浮动元素的容器添加 overflow：hidden；

方法三：使用CSS3的 :after 伪元素

清除浮动其他方法

  1. 父级元素定义 height 。只适合高度固定的布局

  2. 父级元素也一起浮动。不推荐，会产生新的浮动问题
