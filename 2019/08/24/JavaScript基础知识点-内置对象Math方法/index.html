语法

功能

Number

语法

Math.max(num1, num2…numN)

功能

ceil()

语法

Math.ceil(num)

功能

向上取整，即返回大于 num 的最小整数

返回值

Number

floor()

语法

Math.floor(num)

功能

Number

语法

random()

语法

Math.random()

功能

返回值

Number

random=Math.floor(Math.random()*(m-n+1)+n);
