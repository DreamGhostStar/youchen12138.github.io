HTML基础知识点-图像与超链接

图像标签

语法

<img src="" alt=""/>

img属性

属性 值 描述
Src（必写） URL 显示图像的URL
alt 文字 图像替代文本
height 数值和百分比 图像的高
width 数值和百分比 图像的宽

alt属性作用

当由于：

  • 网速太慢

  • src属性中的错误

  • 浏览器禁用图像

用户无法查看图像时，alt属性可以代替图像显示在浏览器中的内容

超链接标签

语法

<a href="">内容</a>

href：链接地址，可以是内部链接或外部链接

当链接为空时，点击链接会刷新页面

属性

属性 描述
href 链接地址
target 链接的目标窗口（_self、_blank、_top、_parent）
title 链接提示文字（当光标停在链接上时，提示文字将会出现）
name 链接命名
