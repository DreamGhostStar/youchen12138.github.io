React

react-redux

理解

  1. 一个 react 插件库

  2. 专门用来简化 react 应用中使用 redux

React-Redux 将所有组件分成两大类

UI 组件

  • 只负责 UI 的呈现，不带任何业务逻辑

  • 通过 props 接收数据（一般数据和函数）

  • 不使用任何 Redux 的 API

  • 一般保存在 components 文件夹下

容器组件

  • 负责管理数据和业务逻辑，不负责 UI 的呈现

  • 使用 Redux 的 API

  • 一般保存在 containers 文件夹下

相关 API

Provider

让所有组件都可以得到 state 数据

connect()

用于包装 UI 组件生成容器组件

mapStateToprops()

将外部的数据（即 state 对象）转换为 UI 组件的标签属性

mapDispatchToProps

将分发 action 的函数转换为 UI 组件的标签属性

简洁语法可以直接指定为 actins 对象或包含多个 action 方法的对象
